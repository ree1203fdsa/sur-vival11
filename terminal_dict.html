📖 터미널 사전 v1.0

터미널에서 사용 가능한 모든 명령어를 정리했습니다.

help
usage: help
현재 터미널에서 사용 가능한 기본 명령어 목록과 설명을 보여줍니다.
ls
usage: ls
시스템에 설치된 모든 앱의 ID 목록을 출력합니다.
open
usage: open [app_id]
특정 앱을 실행합니다. 상점(`win-store`), 메일(`win-mail`) 등을 키워드로 사용할 수 있습니다.
js
usage: js [javascript_code]
실제 자바스크립트 엔진을 구동합니다. `alert()`, `console.log()` 등 모든 JS 기능을 OS 컨텍스트에서 실행할 수 있습니다.
wallpaper
usage: wallpaper [image_url]
전달받은 이미지 URL을 바탕화면 배경으로 즉시 적용합니다.
whoami
usage: whoami
현재 로그인된 유저의 이름, 역할(유저/관리자) 정보를 표시합니다.
clear
usage: clear
터미널의 출력 창을 모두 비우고 초기 화면으로 되돌립니다.
